NAV PILLS CON CONTENIDO

Vista General

Las nav pills ofrecen un estilo alternativo a las tabs tradicionales. Son más redondeadas y modernas.

Son perfectas para interfaces donde quieres un look más suave y menos formal que las tabs estándar.

Características Principales

  • Fácil de implementar
  • Totalmente responsive
  • Personalizable con CSS
  • Compatible con JavaScript de Bootstrap
